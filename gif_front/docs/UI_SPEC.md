# UI 规范（苹果毛玻璃风格）

## 1. 风格关键词
- Glassmorphism（毛玻璃）
- 轻渐变背景 + 半透明卡片 + 细描边 + 柔和阴影
- 字体层级清晰，间距大，圆角偏大（iOS 观感）

## 2. 颜色与材质
- 背景：渐变（冷色系轻盈）
- 卡片背景：`rgba(255,255,255,0.55)`（普通）/ `rgba(255,255,255,0.72)`（强调）
- 描边：`rgba(255,255,255,0.55~0.7)`
- 阴影：`0 16rpx 40rpx rgba(0,0,0,0.08)`
- 主色（按钮/选区）：`#20c05c`
- 文本：
  - 主标题：`rgba(0,0,0,0.9)`
  - 次级：`rgba(0,0,0,0.55~0.7)`

## 3. 组件规范
### 3.1 卡片（工具入口）
- 圆角：`22rpx`
- 内边距：`20rpx`
- 顶部：左侧 badge（胶囊），右侧 tag（轻文字）

### 3.2 结果/操作按钮
- 高度：`96rpx`
- 圆角：`22rpx`
- 字号：`34rpx`，字重 `700`

### 3.3 时间轴（视频转GIF）
- 选区条：底色 `rgba(0,0,0,0.06)`，选中色 `rgba(32,192,92,0.85)`
- 文本：左右时间显示，弱化颜色

## 4. 已落地样式位置
全局毛玻璃样式已在 `app.wxss` 提供：
- `.glass`
- `.glass-strong`
- `.btn-primary`
